Assembly Maven Plugin

Assembly Maven Plugin automates the assembly of projects within the Machanism ecosystem by integrating libraries based on bindex metadata. It streamlines dependency resolution, library selection, and project packaging using GenAI-powered semantic search and metadata-driven workflows.

What this plugin does

This module is packaged as a Maven plugin and exposes the assembly goal. The goal is designed to run without requiring an existing Maven project (i.e. it operates on a configured base directory and is compatible with workflows that generate or modify project content).

High-level workflow

  1. Acquire a prompt. If a prompt file is configured and exists (default: project.txt), its contents are used; otherwise the goal prompts interactively via Plexus Prompter.
  2. Recommend libraries. The plugin invokes the MachAI picker to recommend libraries as bindex entries. Recommendations are logged to the Maven output.
  3. Assemble the project. The MachAI application assembly workflow is run to apply changes to the target directory (e.g., updating a Maven project structure, dependencies, or other generated content).

Configuration

The goal can be configured using Maven properties. Common parameters:

  • assembly.prompt.file — prompt file path (default: project.txt).
  • assembly.score — minimum score threshold for recommendations.
  • bindex.register.url — optional URL used by the picker for metadata registration/lookups.
  • Model/provider selection — the plugin uses separate provider identifiers for:
    • the assembly workflow (ApplicationAssembly model property)
    • the picker workflow (Picker model property)

Usage example

mvn org.machanism.machai:assembly-maven-plugin:assembly

To drive the workflow from a file, place your prompt in project.txt (or configure an alternate file via -Dassembly.prompt.file=...). If the file is absent, the goal will prompt interactively.

Packages

The public API surface for this plugin is small and centered around the single Maven Mojo that implements the goal.

  • org.machanism.machai.assembly.maven — Maven plugin integration for MachAI “assembly” workflows. Provides the assembly goal that reads an assembly prompt, invokes the picker for library recommendations, and runs the assembly workflow on a configured base directory.

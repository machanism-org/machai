Assembly Maven Plugin

Assembly Maven Plugin automates the assembly of projects within the Machanism ecosystem by integrating libraries based on bindex metadata. It streamlines dependency resolution, library selection, and project packaging using GenAI-powered semantic search and metadata-driven workflows.

Overview

This module is packaged as a Maven plugin and provides an assembly goal. The goal is intended to be usable in automation and bootstrap scenarios, and is configured via Maven parameters (typically system properties) rather than relying on an existing project model.

High-level workflow

  1. Acquire a project prompt. If a prompt file is configured and exists (default: project.txt), its contents are used; otherwise the goal asks for input interactively via the Plexus org.codehaus.plexus.components.interactivity.Prompter.
  2. Recommend libraries from metadata. The plugin invokes the MachAI picker to recommend libraries as bindex entries. Recommendations are emitted to the Maven log.
  3. Assemble the project. The MachAI application assembly workflow is executed to apply changes to the configured base directory (for example, updating Maven configuration, dependencies, and generated project content).

Configuration

The goal can be configured using Maven properties. Common parameters include:

  • assembly.prompt.file — prompt file path (default: project.txt).
  • assembly.score — minimum score threshold for recommendations.
  • bindex.register.url — optional URL used by the picker for metadata registration/lookups.
  • Model/provider selection — the plugin uses separate provider identifiers for the assembly workflow and the picker workflow.

Usage

mvn org.machanism.machai:assembly-maven-plugin:assembly

To run using a file-based prompt, create project.txt in the working directory or provide a path via -Dassembly.prompt.file=.... If the file is absent, the goal prompts interactively.

Packages

The public API surface for this plugin is intentionally small and is centered around the single Mojo that implements the goal.

  • org.machanism.machai.assembly.maven — Maven plugin integration for MachAI “assembly” workflows. Provides the assembly goal that reads an assembly prompt, invokes the picker for library recommendations, and runs the assembly workflow on a configured base directory.

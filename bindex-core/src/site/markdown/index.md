## Getting Started
Bindex Core is published on Maven Central. To start, add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>org.machanism.machai</groupId>
    <artifactId>bindex-core</artifactId>
    <version>0.0.2-SNAPSHOT</version>
</dependency>
```

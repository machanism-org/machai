## Getting Started
Bindex Core is available on Maven Central. To integrate, add this dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>org.machanism.machai</groupId>
    <artifactId>bindex-core</artifactId>
    <version>0.0.2-SNAPSHOT</version>
</dependency>
```

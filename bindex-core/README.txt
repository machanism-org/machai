Bindex Core
===========

Project Overview:
-----------------
Bindex Core provides core functionality for bindex metadata management, including generation, registration, library selection, and project assembly. It enables automated handling of library metadata to support efficient discovery, integration, and assembly workflows within the Machanism ecosystem.

Main Features:
--------------
- Generation and registration of project metadata
- Automated library selection and assembly
- Integration with MongoDB and OpenAI models via LangChain4j
- Support for Maven metadata and resolver

Installation:
-------------
This project uses Maven for build and dependency management. To build and install:
1. Install Java 17 or later
2. Run `mvn clean install` in the project directory

Usage:
------
You can use Bindex Core as part of a Machanism-based application or directly from your Java project. Refer to the documentation and JavaDocs for API usage details.

Documentation & Resources:
-------------------------
- Main documentation: `src/site/markdown` (refer to project docs)
- API JavaDocs: Generated with Maven (see `target/site/apidocs` after build)

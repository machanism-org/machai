Bindex Maven Plugin

bindex-maven-plugin enables automated generation and registration of bindex metadata for Maven projects. It facilitates library discovery, integration, and assembly by leveraging structured metadata and GenAI-powered semantic search within the Machanism ecosystem.

What this module provides

This project is packaged as a Maven plugin (maven-plugin) and contributes build goals (Mojos) that generate Bindex metadata for the current Maven project and, optionally, publish that metadata to a registry.

Package overview

The implementation is located in org.machanism.machai.bindex.maven.

  • Concrete goals: {@code create}, {@code update}, {@code register}, and {@code clean}.
  • Shared base Mojo: a common superclass that centralizes Maven integration and parameter handling.

Goals

  • create – creates a new Bindex index for the current project (see {@link org.machanism.machai.bindex.maven.Create}).
  • update – refreshes an existing index (see {@link org.machanism.machai.bindex.maven.Update}).
  • register – scans the project and publishes metadata to a registry (see {@link org.machanism.machai.bindex.maven.Register}).
  • clean – removes temporary artifacts produced by the tooling (see {@link org.machanism.machai.bindex.maven.Clean}).

Execution model

  • Shared configuration: common parameters and Maven integration helpers are implemented by {@link org.machanism.machai.bindex.maven.AbstractBindexMojo}.
  • Optional credential resolution: GenAI credentials may be resolved from Maven settings.xml using a configured server id.
  • Delegation: each goal delegates indexing and registration work to the Machai Bindex Core library.

Typical usage

mvn org.machanism.machai:bindex-maven-plugin:create -Dbindex.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:update -Dbindex.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:register -Dbindex.model=OpenAI:gpt-5 -Dbindex.register.url=http://localhost:8080
mvn org.machanism.machai:bindex-maven-plugin:clean

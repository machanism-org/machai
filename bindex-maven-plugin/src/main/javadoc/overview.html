Bindex Maven Plugin

Bindex Maven Plugin enables automated generation and registration of bindex metadata for Maven projects. It facilitates library discovery, integration, and assembly by leveraging structured metadata and GenAI-powered semantic search within the Machanism ecosystem.

What this module provides

This project is packaged as a Maven plugin (maven-plugin) that contributes build goals (Mojos) for creating and maintaining a Bindex index for the current Maven project, optionally publishing the resulting metadata to a registry.

Package overview

The primary implementation lives in the org.machanism.machai.bindex.maven package, which contains:

  • Goal implementations for creating and updating a Bindex index, registering project metadata, and cleaning up temporary artifacts.
  • A shared base Mojo that centralizes parameter handling and common Maven integration concerns.

Goals

  • create – create a new Bindex index (see {@link org.machanism.machai.bindex.maven.Create}).
  • update – update an existing index (see {@link org.machanism.machai.bindex.maven.Update}).
  • register – scan and publish metadata to a registry (see {@link org.machanism.machai.bindex.maven.Register}).
  • clean – remove temporary artifacts created by the tooling (see {@link org.machanism.machai.bindex.maven.Clean}).

Execution model

  • Module selection: goals may skip modules that are not eligible for indexing (for example, aggregator modules with pom packaging).
  • Shared configuration: common parameters and Maven integration helpers are implemented by {@link org.machanism.machai.bindex.maven.AbstractBindexMojo}.
  • Credentials (optional): GenAI credentials can be resolved from Maven settings.xml using a configured server id.
  • Delegation: each goal delegates indexing and registration work to the Machai Bindex Core library.

Typical usage

The following examples show common command-line invocations. Exact parameters depend on your build configuration and the target registry. 

mvn org.machanism.machai:bindex-maven-plugin:create -Dbindex.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:update -Dbindex.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:register -Dbindex.model=OpenAI:gpt-5 -Dbindex.register.url=http://localhost:8080
mvn org.machanism.machai:bindex-maven-plugin:clean

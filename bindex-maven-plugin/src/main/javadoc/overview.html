Bindex Maven Plugin

Bindex Maven Plugin enables automated generation and registration of bindex metadata for Maven projects. It facilitates library discovery, integration, and assembly by leveraging structured metadata and GenAI-powered semantic search within the Machanism ecosystem.

Module overview

This project is packaged as a Maven plugin (maven-plugin) and exposes Bindex operations as standard Maven goals (Mojos). Each goal runs within a Maven build, reads configuration from the current project, and delegates the heavy lifting to Machai Bindex Core APIs.

Package summary

  • org.machanism.machai.bindex.maven – Maven plugin integration layer for Machai Bindex. Contains the Mojo implementations and shared support code used by all goals.

Goals

The plugin provides the following goals:

  • create – create a new index (see {@link org.machanism.machai.bindex.maven.Create})
  • update – update an existing index (see {@link org.machanism.machai.bindex.maven.Update})
  • register – publish metadata to a registry (see {@link org.machanism.machai.bindex.maven.Register})
  • clean – remove temporary artifacts (see {@link org.machanism.machai.bindex.maven.Clean})

How it works

  • Module eligibility: determines whether the current Maven module should be processed (for example, modules with pom packaging are typically skipped).
  • Configuration: reads build-time parameters from the current project and system properties, including the GenAI provider/model identifier (for example OpenAI:gpt-5).
  • Credentials: can load GenAI credentials from Maven settings.xml when a server id is configured (see {@link org.machanism.machai.bindex.maven.AbstractBindexMojo}).
  • Delegation: invokes Bindex Core operations to create/update indexes and to register project metadata with a registry.

Typical usage

The following examples show common command-line invocations. Exact parameters depend on your build configuration and the target registry. 

mvn org.machanism.machai:bindex-maven-plugin:create -Dgw.genai.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:update -Dgw.genai.model=OpenAI:gpt-5
mvn org.machanism.machai:bindex-maven-plugin:register -Dgw.genai.model=OpenAI:gpt-5 -Dbindex.register.url=http://localhost:8080
mvn org.machanism.machai:bindex-maven-plugin:clean
